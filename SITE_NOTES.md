# Rassvet sale site — first version

The site is a framework-free static page built with HTML, CSS and JavaScript.

## Preview locally

Open `index.html` in a browser. No build step or package installation is required.

## Before public launch

1. Replace the gallery placeholders with 10–14 real high-resolution photographs.
2. Verify every maintenance item against invoices, the ship's log and photographs.
3. Add the asking price and final inventory list.
4. Confirm exact technical specifications, including engine model, displacement, draught and sail areas.
5. Decide whether Telegram is the only public contact method or add an email address.
6. Configure GitHub Pages to deploy from `main` after the pull request is approved and merged.

## Suggested photographs

- Hero shot under sail, landscape orientation.
- Full exterior at the berth from bow quarter and stern quarter.
- Cockpit and deck.
- Salon facing forward and aft.
- Galley, berths and heads.
- Engine compartment and electrical panel.
- Mast, standing rigging and sails.
- Underwater body during haul-out.
- Close-ups of recent work and equipment.

## Editing

Main content lives in `index.html`. Visual styles are in `styles.css`; interaction and contact handling are in `script.js`.
